## WWW & Data Transfer on the Internet

### What is WWW?
- WWW (World Wide Web) is a collection of web pages and resources
- Identified using URLs
- Accessed via browsers like Chrome, Firefox

### Internet vs Web
- Internet = physical + logical network infrastructure
- Web = software layer running on top of the internet

### Data Transfer Basics
- Data is broken into packets
- Packets travel using TCP/IP
- Routers forward packets
- Packets are reassembled at destination

### Core Web Components
- URL: resource address on the web
- HTTP/HTTPS: communication protocol
- HTML, CSS, JavaScript: web content
- Web servers & browsers

### How data flows (High-level)
1. Application generates data
2. Transport layer (TCP/UDP) segments data
3. Network layer adds IP & routes packets
4. Data link & physical layer transmit signals
