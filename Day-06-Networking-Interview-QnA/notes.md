## Networking Interview QnA (Fundamentals)

### Q1. Difference between Internet and WWW?
- Internet: Physical infrastructure (networks, cables, routers).
- WWW: Service on top of internet using HTTP, browsers, websites.

---

### Q2. Explain Client-Server Model
- Client: Requests a service (browser, mobile app).
- Server: Responds with data (web server, API server).

---

### Q3. What is a Port?
- A logical endpoint for communication.
- IP identifies machine, Port identifies application.

Total ports: 0 – 65535

---

### Q4. Common Well-Known Ports
- HTTP: 80
- HTTPS: 443
- FTP: 21
- SSH: 22
- DNS: 53

---

### Q5. What is the primary function of a Router?
- Works at OSI Layer 3 (Network Layer).
- Routes packets between networks using IP addresses.
- Uses routing tables to select best path.

---

### Q6. Difference between Hub, Switch, Router?
- Hub: Sends data to all devices.
- Switch: Sends data to specific MAC address.
- Router: Connects different networks using IP.

---

### Q7. Difference between TCP and UDP?
- TCP: Reliable, connection-oriented, ordered delivery.
- UDP: Fast, connectionless, no delivery guarantee.

---

### Q8. TCP 3-Way Handshake
1. SYN – Client requests connection
2. SYN-ACK – Server acknowledges
3. ACK – Connection established

---

### Q9. IPv4 vs IPv6
- IPv4: 32-bit, limited (~4.3B)
- IPv6: 128-bit, almost infinite

---

### Q10. Public vs Private IP
- Public IP: Assigned by ISP, visible on internet.
- Private IP: Assigned by router, used inside LAN.

---

### Q11. ISP Tiers
- Tier 1: Global backbone providers
- Tier 2: Regional providers
- Tier 3: Local last-mile providers

---

### Q12. What is NAT?
- Maps multiple private IPs to one public IP.
- Saves IPv4 addresses and adds security.
